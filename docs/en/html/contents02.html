Network Visualization¶

Network Visualization Sample
Image Description
sample03:

The left image shows a virtual network topology that was generated in the OpenFlow Network.

  • supported keyboard events

    s: stop the rotation
    left or right: rotation around the y-axis
    up or down: rotation around the x-axis
sample04:

The left image shows a virtual network topology that was generated in the OpenFlow Network.

The sample04 provides following new functions.

  • picking with WebGL
  • displaying the infomation of selected a network device
sample05:

The Left image shows a virtual network topology that was generated in the OpenFlow Network.

The sample05 provides following new functions.

  • visualizing the network traffic in the OpenFlow Network
  • displaying the amount of network traffic
sample06:

The left image shows a virtual network topology that was generated in the OpenFlow Network.

The sample06 provides following new functions.

  • moving network devices
  • rotation around the y-axis of network devices
  • toggle on/off of displaying information
sample07: The left image shows network flows when a scanning utility such as NMAP was executed.
sample08:

The left image shows a virtual network topology that was generated in the OpenFlow Network.

The sample08 provides following new functions.

  • visualizing network topologies from YAML files
  • changing to the network topology which was specified
sample09:

The left image shows network traffics to/from the Internet.

The sample09 provides functions which visualizes the network traffic from the specified information involved of the longitude and latitude.

e.g. from latitude/longitude, to latitude/longitude, packets
[[30.0, 31.2, 35.7, 139.6, 100],
[-25.2, 133.7, 35.7, 139.6, 50],
[56.1, -106.3, 35.7, 139.6, 25]]
// Egypt: (lat=30.0, lon=31.2) Japan: (lat=35.7, lon=139.6)
// Australia: (lat=-25.2, lon=133.7)
// Canada: lat=56.1, lon=-106.3
sample10:

The left image shows a virtual network topology that was generated in the OpenFlow Network.

The sample10 provides following new functions.

  • moving the specified device by the mouse
  • saving position of network devices to YAML files
