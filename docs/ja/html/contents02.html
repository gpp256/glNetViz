WebGL¶

WebGL Samples
イメージ 説明
sample01:

一つのシェーダプログラムを用いて作成したWebGLのサンプルページです。

  • マウスイベントとタッチイベントの両方に対応しました。Android端末(4.0+Chrome Beta)からも表示したオブジェクトの操作が可能です。
sample02: 複数のシェーダプログラムを用いて作成したWebGLのサンプルページです。

Network Visualization¶

Network Visualization Sample
イメージ 説明
sample03:

OpenFlowネットワーク下で形成された仮想ネットワークの構成を見える化してみたサンプルです。

  • キーボードイベントに対応しました。

    s: stop the rotation
    left or right: rotation around the y-axis
    up or down: rotation around the x-axis
sample04:

OpenFlowネットワーク下で形成された仮想ネットワークの構成を見える化してみたサンプルです。

sample03を改造し、次の機能に対応しました。

  • WebGLでのPicking
  • 選択したネットワーク装置が持つ情報の表示
sample05:

OpenFlowネットワーク下で形成された仮想ネットワークの構成を見える化してみたサンプルです。

sample04を改造し、次の機能に対応しました。

  • 通信の流れの見える化
  • 通信量の見える化
sample06:

OpenFlowネットワーク下で形成された仮想ネットワークの構成を見える化してみたサンプルです。

sample05を改造し、次の機能に対応しました。

  • 各ネットワーク装置の配置変更
  • 各ネットワークデバイスのY軸まわりの回転
  • 文字情報の表示/非表示
